# D3Faces Showcase

## Build from sources

```sh
git clone https://github.com/d3faces/d3faces.git
cd d3faces
mvn clean install
cd d3faces-showcase
```

```sh
mvn clean package
```


## Run from sources on [http://localhost:8080/showcase/](http://localhost:8080/showcase/)

```sh
git clone https://github.com/d3faces/d3faces.git
cd d3faces
mvn clean install
cd d3faces-showcase
```

```sh
mvn clean jetty:run
```
